1. The software's file input module shall load the content's of any file stored in secondary storage into main memory given the appropriate file path.

2. The software's file input module shall be able to distinguish text based comma and tab separated file formats from other file formats.

3. The software's file input module shall reject the contents of all files that do not conform to the standards of comma or tab separated file formats.

4. The software's file output modules shall write comma and tab separated files to secondary storage which can then be edited using Microsoft Excel.

5. The software's file output modules shall be able to produce text files that summarize the results of any statistical analysis performed on the data.

6. The software's file output modules shall produce JPEG image files of any chart or graph generated by the software's graphic user interface.
